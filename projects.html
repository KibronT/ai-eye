Winter 2025 Projects

Winter 2025

The inaugural Sprintern cohort produced five projects during the Winter 2025 three-week workshop program. Their work covers AI content detection, browser tools, scam awareness, and AI literacy education.

Winter 2025

AI Scams & Web Development

by Katherine Alvarenga

This project created educational resources to help older adults recognize AI scams and better understand how deceptive technology is used online. It was developed through user feedback, a three-part video series, and website wireframes designed in Figma.

Presentation View Project

Winter 2025

Artificial Intelligence and News Articles

by Navya Gunna

This project analyzes news articles written by humans and AI chatbots to show how synthetic content can imitate real journalism and mislead readers. Using Python, pandas, and sentiment analysis tools, the project compared writing patterns such as tone, sentence structure, and references to dates or official sources.

Presentation View Project

Winter 2025

Bridging Generations Through AI

by Darrell Cox

This project combined beginner-friendly AI education videos with a resume tool that turns life experience into stronger professional resumes. Built with Python, HTML, and VS Code, the resume translator uses role-based logic, action verb banks, and automated bullet generation to create polished resume content.

Presentation View Project

Winter 2025

Validate AI

by Lexie Leong

Validate AI is a browser extension designed to help users verify chatbot responses by checking sources, improving transparency, and reducing misinformation. It was built using JavaScript, HTML, CSS, React, APIs, and JSON, with features like URL validation, prompt engineering, and Google Fact Check integration.

Presentation View Project

Winter 2025

AI Eye

by Sarah Bamba

AI Eye is an interactive quiz concept that helps users practice spotting whether images, videos, or text are real or AI-generated. The project included a working demo website built with HTML and CSS, focused on clear navigation, accessibility, and simple learning tools.
