What is it?

Validate AI is a Chrome extension designed to help users better understand and trust AI-generated responses. When an AI provides an answer, the extension checks whether the sources it cites actually exist, evaluates the credibility of those sources, and looks for similar claims that have already been fact-checked by trusted organizations. By combining source validation, thoughtful prompt design, and fact-checking tools, the extension gives users more transparency and control, helping them decide when an AI response is reliable and when it should be questioned.
